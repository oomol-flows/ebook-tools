# 电子书格式转换器

一个全面的电子书格式转换工具，支持在 EPUB、MOBI、AZW3、PDF 等流行电子书格式之间进行转换。该工具基于可靠的 Calibre 引擎构建，为各种电子书格式提供易于使用的转换模块。

## 🎯 工具功能

本项目提供一套转换工具，可以将您的电子书从一种格式转换为另一种格式，使其与不同的阅读设备和应用程序兼容。无论您是想在其他设备上阅读 Kindle 书籍，还是将 PDF 转换为更适合阅读的格式，这个工具都能满足您的需求。

## 📚 可用转换模块

### 1. 通用电子书转换器
**支持多种格式的最通用转换器**

- **输入格式**：EPUB、MOBI、AZW、AZW3、PDF、TXT、HTML、FB2、RTF
- **输出格式**：EPUB、MOBI、AZW3、PDF、TXT、HTML
- **特色功能**：
  - 质量设置（低、中、高、最佳）
  - 设备优化（Kindle、iPad、Kobo、Nook 等）
  - 元数据保留
  - 为高级用户提供的自定义选项

### 2. EPUB 转 AZW3 转换器
**针对 Kindle 设备优化**

- 将标准 EPUB 文件转换为亚马逊的 AZW3 格式
- Kindle 设备优化（Paperwhite、Oasis、Fire 等）
- 高质量转换并保留元数据
- 完美支持在 Kindle 设备上阅读 EPUB 书籍

### 3. EPUB 转 MOBI 转换器
**经典 Kindle 格式支持**

- 将 EPUB 文件转换为传统的 MOBI 格式
- 兼容旧版 Kindle 设备
- 可调节质量设置
- 保留书籍元数据和结构

### 4. AZW3 转 EPUB 转换器
**将亚马逊书籍转换为通用格式**

- 将 AZW3/AZW 文件转换为广泛支持的 EPUB 格式
- DRM 保护文件处理（有限支持）
- 格式清理和优化
- 使亚马逊书籍可在非 Kindle 设备上阅读

### 5. MOBI 转 EPUB 转换器
**传统格式到现代格式转换**

- 将 MOBI/AZW 文件转换为 EPUB 格式
- 格式修复以获得更好的兼容性
- 质量保留选项
- 现代化旧电子书格式

## 📖 工作原理

该工具使用 **Calibre**（一款功能强大且成熟的电子书管理软件）作为其转换引擎。Calibre 多年来一直是电子书转换的黄金标准，支持数十种格式的高保真转换。

### 转换过程
1. **文件分析**：工具检查您的输入文件以了解其格式和结构
2. **格式转换**：Calibre 转换内容同时保留文本、图像和格式
3. **优化**：内容针对您的目标设备或格式进行优化
4. **质量保证**：验证转换并生成详细日志
5. **输出交付**：转换后的文件保存到您指定的位置

## 💡 获得最佳效果的技巧

### 选择合适的转换器
- 对于大多数转换任务使用**通用电子书转换器**
- 对于特定格式对（如"EPUB 转 AZW3"）使用专用转换器以获得优化结果
- 选择质量和优化设置时考虑您的目标阅读设备

### 质量设置指南
- **低**：最快转换、最小文件大小、基本质量
- **中**：速度、大小和质量的良好平衡（推荐给大多数用户）
- **高**：更好的质量、更长的转换时间、更大的文件
- **最佳**：最高质量、最慢转换、最大文件

### 格式推荐
- **EPUB**：最适合大多数现代电子阅读器和应用（除 Kindle 外）
- **AZW3**：最适合现代 Kindle 设备
- **MOBI**：最适合旧版 Kindle 设备
- **PDF**：最适合复杂布局的文档（不推荐用于小说）

## 🔒 重要说明

- **DRM 限制**：此工具无法转换受版权保护的电子书。仅用于您合法拥有并有权转换的电子书。
- **质量变化**：转换质量取决于源文件质量和格式复杂性。
- **文件安全**：转换前请始终保留原始文件的备份。
